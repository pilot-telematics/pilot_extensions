🌍 Hello World Application

Basic example application for PILOT Extensions

🚀 Purpose

This is the simplest possible PILOT Extensions application designed to help developers understand the basic structure and functionality.

🎯 Features Demonstrated

  • Creating navigation tabs
  • Adding content to main area
  • Adding buttons to header
  • Loading custom styles
  • Basic Ext JS component usage

🛠️ Technical Details

PILOT Extension Name: hello_world

Main Class: Store.hello_world.Module

Extends: Ext.Component

Entry Point: initModule() function

📁 File Structure

hello_world/
├── index.html     # Root file for Cloudflare assets root
├── Module.js      # Main application file
├── doc/
│   └── index.html # This documentation
└── style.css      # Custom styles

🎓 Learning Path

  1. Start with this Hello World example
  2. Study the airports example for map integration
  3. Explore planets for data handling without maps
  4. Use template_app as a base for your own projects

Perfect for developers taking their first steps with PILOT Extensions!
